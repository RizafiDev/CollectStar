### Instalasi

1. Clone repository:
```bash

```

2. Masuk ke direktori proyek:
```bash

```

3. Install dependencies:
```bash

```

4. Jalankan development server:
```bash
npm start
# atau
yarn start
```

5. Buka browser Anda dan kunjungi `http://localhost:3000` untuk bermain!
